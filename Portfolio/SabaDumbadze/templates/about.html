ვინ ვარ მე

მე ვარ ვებ-დეველოპერი საქართველოდან. პროგრამირებით მას შემდეგ დავინტერესდი, რაც გამიჩნდა სურვილი ამეწყო ისეთი საიტები რომელსაც ყოველდღე გამოვიყენებდი. რაც თავიდან უბრალო ჰობი იყო, მალე დიდ ინტერესად იქცა და დღეს ნულიდან სხვადასხვა სისტემების აწყობა ჩემი საყვარელი ჰობიც და საქმეცაა.

მიუხედავად იმისა, რომ ჯერ კიდევ თინეიჯერი ვარ, ჩემს საქმეს სრული პასუხისმგებლობით ვუდგები. დროს ძირითადად იმაზე ვხარჯავ, რომ ვისწავლო, როგორ გავხადო ვებგვერდები უფრო სწრაფი და მარტივად გამოყენებადი. ჩემთვის მთავარია არა მხოლოდ ვიზუალი, არამედ ის, თუ რამდენად გამართულად მუშაობს კოდი და რამდენად კომფორტულია საიტი მომხმარებლისთვის.

მუშაობის სტილი

მიყვარს სიმარტივე. იქნება ეს Django-ზე ბექენდის აწყობა თუ CSS-ით საიტის ვიზუალზე მუშაობა, ვცდილობ კოდი ისე დავაორგანიზო, რომ მისი მართვა მომავალშიც ადვილი იყოს. მჯერა, რომ კარგი ვებგვერდი არ უნდა იყოს ზედმეტად გადატვირთული, არამედ ის უნდა იყოს სუფთა, გასაგები და იდეალურად მორგებული მობილურ მოწყობილობებზე.

უნარები და განვითარება

მაქვს გამოცდილება Python-სა და Django-ში, ასევე თავისუფლად ვმუშაობ HTML, CSS და JavaScript ტექნოლოგიებით. პროექტის დაგეგმვისთვის ვიყენებ Figma-ს, ხოლო ვერსიების კონტროლისთვის — Git-ს. ვცდილობ მუდმივად მივდიო სიახლეებს, რადგან დეველოპმენტში ყოველდღე რაღაც ახალი ჩნდება და არ მინდა სტაგნაცია.

ჩემი მიზანი

მინდა დავეხმარო ადამიანებსა და მცირე ბიზნესებს ხარისხიანი ონლაინ სივრცის შექმნაში. მსიამოვნებს ისეთ პროექტებზე მუშაობა, სადაც შემიძლია რაღაც ახალი ვისწავლო და საბოლოოდ ისეთი პროდუქტი ჩავაბარო დამკვეთს, რომლის ჩვენებაც სხვებისთვის არ შერცხვება. თუ გაქვს იდეა, შემიძლია დაგეხმარო მის რეალობად ქცევაში.
